Publish an InDesignAuthoring software from Adobe document online

  1. Open an InDesignAuthoring software from Adobe document and select File > Publish Online. Export > Save as pdf.

  2. Select the General tab to add the following information:

     
    • Title: Enter the name of the online document. This is also the browser title of the published output.
    • Description: Enter the description of the online document. You'll view the description in the Publish Online dashboard and on any social media forums.
    • Pages: Select to export All or a Range of pages from the current document.
    • Export As: Select Single or Spread to specify the display settings.
    • Enable Password Protection: By default, password protection is disabled. Toggle on to set a password for your Publish Online document. Only viewers with the password can access the document.
     

    If you are republishing an existing document, select Update Existing Document and select the app.
